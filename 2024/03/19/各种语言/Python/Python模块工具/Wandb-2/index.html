wandb.agent() 是用于启动和管理 Sweep 中的实验执行的一个函数，接收两个主要参数：

  • sweep_id：这是你之前创建或通过 API 获取到的 Sweep 的唯一标识符
  • function：这是一个可调用对象，通常是一个函数，这个函数负责执行实际的训练逻辑，并且可以访问当前 sweep 运行时传入的超参数
